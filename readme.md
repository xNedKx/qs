# 測驗程式

提供使用者測驗與紀錄的測驗程式。

預載了六個中醫藥相關的題庫，支援多選以及單選兩種題型。

並提供自行建立題庫以及題庫的匯入與匯出。

## 測驗說明

左方有題庫選擇與編輯器開關，右方有題目清單。

回答記錄會以(正確/錯誤)的格式顯示在題目清單上，最後一次回答若為正確，背景將為綠色，錯誤為紅色。

顯示答案時，未被選擇的正確答案為藍色，正確的選擇為綠色，錯誤的選擇則顯示為紅色。

離開頁面時會自動保存紀錄，或可以手動進行儲存。

## 編輯器說明

每個題庫可以有多個題組，每個題組內的題目會從選項清單中自動取出數個作為題目選項，並會避開數值相同的選項。

題庫的參數主要有三個：代碼、名稱與參數。代碼請使用英數並避免符號與空格，名稱為顯示在題庫裡的文字，參數則為進階設定使用。

每個題組的參數則有四個：名稱、選項代碼、題型、選項數量。請避免在不同題組使用相同名稱或代碼，這可能會使前面的題組被後面的覆蓋。題型預設提供全選及擇一，全選代表全部的答案都要符合，為多選題；擇一則只要其中一個答對便可，為單選題。選項數量請設定超過題目答案的最高數量，否則將發生錯誤。

題目的資料欄位有四個：名稱、答案、描述與不建立。名稱欄位為顯示在清單裡的名稱，另外它會替代描述中的{name}字串。答案請一行輸入一個項目，並且避免使用半型逗號(,)與分號(;)，所有的答案將會自動加入選項，除非勾選了不加入方塊。描述則是顯示在題目位置的問題描述。

選項的資料欄位有三個：選項數值、顯示為與排除。選項數值可以說是選項的代號，與題目中的答案一致時便會視為正確。顯示為欄位則是這個選項顯示上的文字，舉例來說，一個選項可以數值為水果，顯示上為蘋果，對於一個問說以下何者為水果的題目，它會是正確的答案。排除則是一個簡易避免混淆選項出現用的欄位，在題目選用了某個選項後，便會把數值等同於排除的選項排除掉，讓題目裡不會同時出現這些選項，排除欄位跟題目的答案欄位一樣，一行輸入一個要排除的項目。

在題目與選項的清單中，可以使用 ctrl + 方向鍵移動游標，而在非多行的欄位上按 Enter 將會自動建立新行；當游標在第一欄且該欄位沒有資料的時候，按 backspace 將會移除該行資料。

題目與選項均可以獨立匯入匯出，整個題庫也可以保存於瀏覽器中，或是匯出成檔案。
